PetsCoop: Sistema de agendamento para petshop

Projeto de conclusão da materia LP4, baseado no projeto de agendamento de corridas desenvolvido em aula, aplica arquitetura MVC, PHP e MySQL.

FUNCIONALIDADES:
    Area do Cliente: A home seria a area do cliente onde ele preenche um formulário de agendamento de serviço, escolhe entre os 3 disponíveis:banho, tosa e atendimento veterinario. Faz então um upload da foto do pet que entra pra lista de agendamento.
    
    Area do funcionario: Um painel que é acessado via login e senha que permite a vizualização e conclusão dos atendimentos.

    Interface construida com Bootstrap, com cores customizadas.

TECNOLOGIAS:
    Backend: PHP.
    Banco de Dados: MySQL.
    FrontEnd: HTML5, CSS3, Bootstrap 5.
    Arquitetura: MVC
